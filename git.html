Hogyan készül?

A Git repository készítése egyszerű, a GitHubon pár kattintással meg lehet csinálni.

Bejelentkezés után a repository fülnél (több helyen is el lehet érni, legegyszerűbb ha a profilnál kiválasztjuk a repository-t) és a new gombra kattintva már csak egy nevet megadva tudjuk létrehozni a repót.

A létrehozásnál több dolgot is ki tudunk választani, legegyszerűbb ha public-on hagyjuk és hozzáadunk egy README file-t.
